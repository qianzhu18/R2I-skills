# Overview

## What This Repository Is

`Aut_Sci_Write` is a modular academic skill suite, not a single product application. It packages six Claude Code research skills that cover literature search, PDF insight extraction, figure extraction, review writing, Zotero integration, and presentation generation.

## Who It Serves

- graduate students
- researchers
- users running academic workflows in Claude Code

## Why It Matters

The main value is workflow compression: instead of bouncing between separate paper-search, extraction, citation, and slide tools, the user gets skill-native workflows inside one agent environment.

## Confidence Level

High confidence:
- this is a multi-skill research toolkit
- `sci-ppt` is the most integrated workflow
- Python is the main implementation language

Lower confidence:
- exact production adoption
- how heavily each skill is used in practice
