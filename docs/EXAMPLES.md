# Examples

Examples are regression fixtures for the learning-document workflow.

## Current Example Set

### `aut-sci-write-docs/`

A multi-page docs bundle for the `Aut_Sci_Write` repository.

Use it to test:

- docs-bundle navigation
- study-doc page completeness
- preview server behavior
- validator behavior

### `aut-sci-write-study-doc.md`

A single-file fallback example.

Use it to confirm the preview server can still render a compact markdown path when a docs bundle is not available.

## What A Good Example Should Prove

A strong example should help us test at least one of:

- skill suite repositories
- frontend application repos
- backend service repos
- CLI or tooling repos
- generated-output traps
- contribution vs learner vs interviewer reading lanes
